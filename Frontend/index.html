The backend supports two roles: artist can upload music and create albums, and user can browse music and albums.

Auth routes

  • POST /api/auth/register
  • POST /api/auth/login
  • POST /api/auth/logout

Artist routes

  • POST /api/music/upload
  • POST /api/music/album

User routes

  • GET /api/music
  • GET /api/music/albums
  • GET /api/music/albums/:albumId
